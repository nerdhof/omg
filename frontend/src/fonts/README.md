# 39C3 Fonts

This directory should contain the font files from the 39C3 style guide.

## Download Fonts

Download the latest design package from:
https://events.ccc.de/congress/2025/infos/styleguide.html

## Required Font Files

Place the following font files in this directory:

### KarioDuplexVar
- `Kario39C3Var-Roman.ttf`
- `Kario39C3VarWEB-Roman.woff`
- `Kario39C3VarWEB-Roman.woff2`

### OfficerSansCond
- `OfficerSansCondWeb-Regular.woff`
- `OfficerSansCondWeb-Regular.woff2`
- `OfficerSansCondWeb-Bold.woff`
- `OfficerSansCondWeb-Bold.woff2`
- `OfficerSansCondWeb-Italic.woff`
- `OfficerSansCondWeb-Italic.woff2`
- `OfficerSansCondWeb-BoldItalic.woff`
- `OfficerSansCondWeb-BoldItalic.woff2`

### OfficerSans
- `OfficerSansWeb-Regular.woff`
- `OfficerSansWeb-Regular.woff2`
- `OfficerSansWeb-Bold.woff`
- `OfficerSansWeb-Bold.woff2`
- `OfficerSansWeb-Italic.woff`
- `OfficerSansWeb-Italic.woff2`
- `OfficerSansWeb-BoldItalic.woff`
- `OfficerSansWeb-BoldItalic.woff2`
